CSS 文本属性

以下每个卡片演示一个 CSS 文本属性，拖动滑块或选择下拉菜单可以实时查看效果

color 设置文本颜色
这是一段彩色文字，试试改变我的颜色吧！
#4f46e5
color: #4f46e5;
direction 设置文本方向
→ 从左到右（LTR）Hello World，你好世界！
direction: ltr
direction: ltr;
letter-spacing 设置字符间距
每一个字符之间的距离
0px
letter-spacing: 0px;
line-height 设置行高
行高控制的是行与行之间的垂直间距。
这是第二行文字，当行高值变大时，两行之间的距离会增大。
试试拖动滑块调整行高，观察每一行之间的空隙变化。
1.6
line-height: 1.6;
text-align 对齐元素中的文本
这段文字的对齐方式会随着你的选择而变化。text-align 控制文本在容器中的水平对齐方式，常用值有 left、center、right、justify。
text-align: left
text-align: left;
text-decoration 向文本添加修饰（下划线、删除线等）
这段文字可以添加下划线、上划线或删除线
text-decoration: none
text-decoration: none;
text-indent 缩进元素中文本的首行
这段文字的第一行设置了首行缩进。首行缩进在中文排版中很常见，通常段落首行会缩进 2 个汉字（约 2em）。调整滑块看看效果，注意只有第一行被缩进，后面的行不受影响。
0px
text-indent: 0px;
text-shadow 设置文本阴影
文字阴影效果
2px 2px 4px #888888
text-shadow: 2px 2px 4px #888888;
text-transform 控制元素中的字母大小写
hello world! css is fun. 中文不受影响。
text-transform: none
text-transform: none;
unicode-bidi 与 direction 配合，控制双向文本的覆盖行为
ABC 你好 123 DEF 世界 456
unicode-bidi: normal
unicode-bidi: normal;
vertical-align 设置元素的垂直对齐方式（行内元素或表格单元格）
X下标X （关注「下标」二字的位置变化）
vertical-align: baseline
vertical-align: baseline;
white-space 设置元素中空白（空格、换行）的处理方式
这段文字包含 多个空格 和 换行。 不同 white-space 值会以不同方式处理它们。
white-space: normal
white-space: normal;
word-spacing 设置单词之间的间距
Hello World! CSS is awesome. 中文 词语 间距 也受 影响。
0px
word-spacing: 0px;
